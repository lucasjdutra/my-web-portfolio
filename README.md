Este é o repositório do meu portfolio profissional de design e desenvolvimento de software. Um projeto de página única (SPA), focado em experiência do usuário (UX/UI).

---

## ✨ Características Principais

- **🌐 Mudança de Idiomas**: Alternância entre **Inglês** e **Português (BR)**, permitindo alcance global para recrutadores de qualquer localidade.
- **📜 Rolagem Suave**: Integração do **Lenis Scroll** para suavização da experiência de rolagem no navegador.
- **📱 Responsividade"**: Layout responsivo para qualquer tamanho de tela, desde celulares até monitores de alta resolução.
- **🎨 Design e Tipografia**: Escolha intencional de cores de alto contraste, tipografia elegante e uso do espaço negativo.

---

## 🛠️ Tecnologias e Bibliotecas

- **React 18 & TypeScript**
- **Tailwind CSS v4**
- **Vite**
- **Lenis Smooth Scroll**
- **Lucide React**

---

## 📂 Estrutura do Projeto

```text
├── src/
│   ├── img/            # Ativos visuais e mockups dos projetos
│   ├── components/     # Componentes estruturais e isolados de UI
│   ├── App.tsx         # Componente principal do portfólio (Fluxo de dados, traduções, seções)
│   ├── main.tsx        # Ponto de entrada do React
│   ├── index.css       # Configurações globais de CSS e importações do Tailwind CSS v4
│   └── vite-env.d.ts   # Declarações globais de tipos do Vite
├── index.html          # Template HTML principal
├── tsconfig.json       # Configuração de tipagem estrita do TypeScript
└── vite.config.ts      # Configurações de build do Vite
```
